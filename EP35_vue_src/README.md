# vue3-demo


前置步骤 

 -1. 先完成智能合约的编写
 -2. 创建前端Vue项目

第一步  先引入web3.js  
//cdn.jsdelivr.net/npm/web3@latest/dist/web3.min.js

第二步  将合约的abi数组复制粘贴到vue3项目 保存为 .json文件


第三步  你的浏览器需要安装 metamask 插件


第四步  开始编写前端代码
